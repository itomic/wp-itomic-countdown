# Itomic Countdown WordPress Plugin - Project Overview

## Project Summary

A professional WordPress plugin that displays real-time countdown timers to any event on your website. Currently at version 1.0.10 with automated deployment pipeline and comprehensive development environment.

## Key Features

### Core Functionality
- **Real-time countdown**: Displays days, hours, minutes, and seconds until your event
- **Customizable display**: Choose from 9 different positions on your site
- **Timezone support**: Automatic timezone detection with manual override options
- **Admin interface**: Easy-to-use settings page in WordPress admin
- **Auto-updates**: Self-hosted update system with automatic notifications
- **Responsive design**: Works on all devices and screen sizes
- **WordPress standards**: Follows WordPress coding standards and best practices

### Technical Features
- Modern CSS with backdrop-filter effects
- JavaScript real-time updates every second
- Timezone-aware calculations
- Input sanitization and validation
- Output escaping for security
- Nonce verification for forms
- Capability checking for admin access

## Display Positions

The countdown can be displayed in 9 different positions:
- **Top**: Left, Middle, Right
- **Middle**: Left, Middle, Right  
- **Bottom**: Left, Middle, Right
