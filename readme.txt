== Description ==

Itomic Countdown is a lightweight, easy-to-use WordPress plugin that allows you to display a real-time countdown timer to any event on your website.

**Features:**

* **Easy Setup**: Simple configuration through WordPress admin panel
* **Real-time Updates**: Live countdown that updates every second
* **Multiple Positions**: Choose from 9 different display positions (top, middle, bottom Ã— left, center, right)
* **Timezone Support**: Full timezone support for accurate countdown display
* **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
* **Modern Styling**: Beautiful, modern design with hover effects
* **Accessibility**: Built with accessibility in mind, supporting reduced motion preferences

**Perfect for:**
* Product launches
* Event announcements
* Sales countdowns
* Holiday countdowns
* Conference deadlines
* Any time-sensitive event

**How to Use:**

1. Install and activate the plugin
2. Go to Settings > Itomic Countdown
3. Enter your event title
4. Select your event date and time
5. Choose your timezone
6. Select your preferred display position
7. Save settings and view your site!

The countdown will automatically appear on your website and update in real-time.

== Installation ==

1. Upload the `itomic-countdown` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Itomic Countdown to configure your countdown
4. Enter your event details and save

== Frequently Asked Questions ==

= Does this plugin work with all themes? =

Yes! The countdown is displayed as a fixed overlay that works with any WordPress theme.

= Can I have multiple countdowns? =

Currently, the plugin supports one countdown per site. Multiple countdown support is planned for future versions.

= Is the countdown mobile-friendly? =

Absolutely! The countdown is fully responsive and adapts to all screen sizes.

= Can I customize the styling? =

Yes, you can override the CSS styles in your theme's stylesheet or use custom CSS plugins.

= What happens when the countdown reaches zero? =

The countdown displays "Event has started!" when the target time is reached.

= Does this plugin affect my site's performance? =

No, the plugin is lightweight and optimized for performance. It only loads the necessary files when a countdown is configured.
